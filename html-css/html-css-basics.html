Selectors

Multifunctional

  • * {} all elements
  • p + a {} siblings (after)
  • p ~ a {} all a after p
  • p > a {} all children a of parent p

Attributes

  • p[attr] {}
  • p[attr="smth"] {}
  • p[attr$="smth"] {} ends with 'smth'
  • p[attr^="smth"] {} begins with 'smth'
  • p[attr*="smth"] {} contains 'smth'
  • p[attr~="smth"] {} contains a separate word 'smth'
  • p[attr|="smth"] {} - and 'smth'

Link's pseudo-classes

  • a:link {} unvisited, always first
  • a:visited {} always second
  • anyselector:hover {} always third
  • anyselector:active {} always forth
  • anyselector:target {} on link's anchor target

Pseudo-classes by count

Counts all parent's selectors

  • anyselector:first-child {}
  • anyselector:last-child {}
  • anyselector:only-child {}
  • anyselector:nth-child(even) {}
  • anyselector:nth-child(odd) {}
  • anyselector:nth-child(2) {}
  • anyselector:nth-child(an+b) {} a = repeats for n, b = start point
  • anyselector:nth-last-child(an+b) {} counts from last
  • anyselector:nth-child(odd) {}
  • anyselector:nth-child(odd) {}
  • anyselector:nth-child(odd) {}

Counts only same parent's selectors

  • anyselector:only-of-type {}
  • anyselector:nth-of-type(even) {}
  • anyselector:nth-of-type(odd) {}
  • anyselector:nth-of-type(3) {}
  • anyselector:nth-of-type(an+b) {}
  • anyselector:nth-last-of-type(an+b) {}

:not pseudo-class

  • anyselector:not(anyselector) {}

Pseudo-elements

Always inside the container, can't use on pseudo-classes

  • anyselector::first-letter {}
  • anyselector::before { content: '' }
  • anyselector::after { content: '' }
