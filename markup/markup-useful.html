Markup: Useful

Setup tips

  • Add min-width to main blocks (header, footer, main, container) for content not to be cut on scroll or resize
  • Place popups before scripts at the end of a html, place overlay block next to popup
  • Container - for sizes, settings
  • Wrapper - only for backgrounds

Fonts

Font weights

  • 100 = thin (hairline)
  • 200 = extra light (ultra light)
  • 300 = light
  • 400 = normal
  • 500 = medium
  • 600 = semi bold (demi bold)
  • 700 = bold
  • 800 = extra bold (ultra bold)
  • 900 = black (heavy)

Complex properties order

It is better to use separate properties in selectors, but the order has to be like in a complex one

  • font: style, variant, weight, size/line-height, family
  • background: color, image, position/size, repeat, origin, clip, attach, initial|inherit
